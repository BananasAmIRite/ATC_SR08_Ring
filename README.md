# SR08 Smart Ring Gesture Recognition

- Streams accelerometer data at 10 Hz via Bluetooth from an SR08 ring
- Runs gesture recognition via ML5js on accelerometer data (with preprocessing) to determine idle vs circle clockwise vs circle counterclockwise
- Bounds gesture to native windows controls (volume up, volume down)
- Includes dedicated trainer to train different gestures & export to files
